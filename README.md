# Authentication
## Members Only
### A Rails project using Rail's Authentification & Model Relationships
#### Project 6

## Getting Started

To get started with the app, clone the repo and then install the needed gems:

```
$ bundle install --without production
```

Migrate the DB:

```
$ rake db:migrate
```

Run the Rails Server

```
$ rails server
```
